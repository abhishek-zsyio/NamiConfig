# 💻 Cross-Platform Key Guide

| Key        | Description                | Platform |
| :--------- | :------------------------- | :------- |
| `<leader>` | Space bar                  | All      |
| `<C-...>`  | Ctrl + ...                 | All      |
| `<A-...>`  | Option (Mac) / Alt (Linux) | All      |
| `<S-...>`  | Shift + ...                | All      |

## 🧭 Core & Nav

| Key          | Action                    |
| :----------- | :------------------------ |
| `<leader>ch` | Open this cheat sheet     |
| `jk`         | Escape Insert Mode        |
| `;`          | Command Mode (like `:`)   |
| `<Esc>`      | Clear Highlights          |
| `<C-s>`      | Save File                 |
| `<C-q>`      | Quit Neovim               |
| `<leader>th` | Select Theme              |
| `<leader>n`  | Toggle Line Numbers       |
| `<leader>tc` | Toggle Treesitter Context |

## 📁 File Explorer

| Key          | Action                     |
| :----------- | :------------------------- |
| `<leader>e`  | Open / Focus File Explorer |
| `<C-n>`      | Toggle File Explorer       |
| `<leader>ff` | Find Files                 |
| `<leader>fw` | Find Words (Grep)          |
| `<leader>fr` | Recent Files               |
| `<leader>fb` | Find Buffers               |
| `<leader>gL` | Git Log / Blame            |

## 🪟 Window & Buffer

| Key           | Action                         |
| :------------ | :----------------------------- |
| `<C-h,j,k,l>` | Move Left, Below, Above, Right |
| `<Tab>`       | Next Buffer                    |
| `<S-Tab>`     | Previous Buffer                |
| `<leader>x`   | Close current buffer           |

## 🖥️ Terminal

| Key      | Action                     |
| :------- | :------------------------- |
| `<C-\`>` | Toggle Floating Terminal   |
| `<A-h>`  | Toggle Horizontal Terminal |
| `<A-v>`  | Toggle Vertical Terminal   |

## 🛠️ Code Editing & LSP

| Key          | Action                           |
| :----------- | :------------------------------- |
| `<leader>/`  | Toggle Comment (Lines)           |
| `<A-j/k>`    | Move selected lines Down / Up    |
| `<leader>fm` | Format File                      |
| `gd`, `gD`   | Go to Definition, Declaration    |
| `gi`, `gr`   | Go to Implementation, References |
| `K`          | Hover Documentation              |
| `<leader>ca` | Code Action                      |
| `<leader>ra` | Rename Symbol                    |
| `<leader>ds` | LSP Diagnostics                  |

## 🏆 LeetCode

| Key          | Action                                |
| :----------- | :------------------------------------ |
| `<leader>lc` | Open LeetCode Dashboard / Console     |
| `<leader>lr` | Save & Run Test Cases Asynchronously  |
| `<leader>ls` | Save & Submit Solution Asynchronously |
| `<leader>ld` | Open Problem Description Panel        |
| `<leader>li` | Show Problem Info & Stats             |
